Access indexes in arrays

Items in arrays are accessed by indexes. The first item is located in index 0, the second in index 1, and so on. Index 0 would be equal to 1, and index 1 would be equal to 2.

array = [1,2,3,4,5]

The index can be access by using the following code:

array[index number]

To access the value of "2" the code would be:

array[1]

Try It

Modify an array with indexes

array = [1,2,3,4,5]

An index in an array can be modified by declaring the index of an array and specifying its new value.

array[3] = 8

This will change the value of 4 to 8.

Try It

array = [1,2,3,4,5]

Access multi-dimensional arrays

Continue to add indexes in brackets to access arrays within arrays.

array = [[1,2,3], [4,5,[6,7]]]

For example, to access the second array ([4,5,[6,7]]) use:

array[1]

To access the third item in the second array ([6,7]) use:

array[1][2]

To access the value 6 use:

array[1][2][0]

Try It

e.g. Enter [1][2] as 1 2

Add items to an array

array = [1,2,3]

An item can be added to the end of an array with push() and the beginning of an array with unshift().

array.push(4)

This will change the array to [1,2,3,4]

array.unshift(4)

This will then change the array to [4,1,2,3,4]

Try It

array = [1,2,3]

Remove items from array

array = [1,2,3,4,5,6,7]

An item can be removed from the end of an array with pop() and from the beginning of an array with shift().

array.pop()

This will change the array to [1,2,3,4,5,6]

array.shift()

This will then change the array to [2,3,4,5,6]

Try It

array = [1,2,3,4,5,6,7]

If Else vs Switch

In this example we will use an if else statement to accomplish the same task as switch. We want to determine a person's hair color based on the number entered:

0 = black    1 = brown    2 = blonde    3 = red

If Else Statement


if (color == 0) {
    colorIfElse = "black";
} else if (color == 1) {
    colorIfElse = "brown";
} else if (color == 2) {
    colorIfElse = "blonde";
} else if (color == 3) {
    colorIfElse = "red";
} else {
    colorIfElse = "not specified";
}
Switch Case Statment

To use Switch Case below, colorNumber must be typeof number. Cases can be strings instead, "0".

switch(colorNumber) {
    case 0: colorSwitch = "black";
    break;
    case 1: colorSwitch = "brown";
    break;
    case 2: colorSwitch = "blonde";
    break;
    case 3: colorSwitch = "red";
    break;
    default: colorSwitch = "not specified";
    }

Do While Loop

A Do While loop will ensure that the code inside the loop will execute at least once, since the while check is located at the end of the loop. The Do While loop below will result in an array of [1].
var array = [];
i = 1;
do {
   array.push(i);
   i++;
} while (i <= 1);

parseInt("string", base)

Generate Random Numbers

Math.random()

Math.random() will generate a random decimal number between 0 and 1 which includes 0 but not 1.

Math.floor()

Math.floor() will round the decimal down.
Math.random() and Math.floor() can be combined to generate a whole number between 0 and a second number by nesting the functions and multiplying by a number. Because Math.random() does not include the upper number, the number multiplied must be one above the desired upper value. The function below will generate a number between 0 and 10.

Math.floor(Math.random() * 11)

A range can be specified by using the following. The min and max values will be included.

Math.floor(Math.random() * (max - min + 1) ) + min

To generate a number between 2 and 5, the code will look like this:

Math.floor(Math.random() * (5 - 2 + 1) ) + 2

Math.floor(Math.random() * 4) + 2

This will generate a number from a starting point of 2 and counting up until a total of 4 numbers is reached, so 2, 3, 4, 5.

Conditional/Ternary Operator

An if else statement can be written using the conditional/ternary operator.
if (condition) {
   return "statement if true";
} else {
   return "statement if false";
}
condition ? "statement if true" : "statement if false"
The conditional/ternary operator can also be used for if / else if statements.
if (condition1) {
   return "statement if true";
} else if (condition2) {
   return "this statement if true";
} else {
   return "this statement all false";
}
  condition1 ? "statement if true"
: condition2 ? "this statement if true"
: "this statement all false"

var vs let

var allows declaring a variable more than once. This is acceptable but causes a problem if this was not done intentially.
var animal = "dog";
var animal = "cat";
ES6 introduces the variable "let". If a let variable is declared more than once within the same scope, an error will be thrown in the console.log.
let animal = "dog";
let animal = "cat";
However, the same let variable can be declared within different scopes without affecting the value of the variable outside of the scope it is declared.
let animal = "dog";
function animalType() {
   let animal = "cat";
   console.log("Animal inside function:", animal);
}
animalType();
console.log("Animal outside function:", animal);

const vs let

Once a let variable has been declared, it can be changed or reassigned.
let animal = "dog";
function animalType() {
   let animal = "cat";
   animal = "snake";
   console.log("Animal inside function:", animal);
}
animalType();
animal = "bird";
console.log("Animal outside function:", animal);
const has the same features as let; however, once const has been declared, it cannot be reassigned. Good practice is to name constants in all CAPS.
const PI = 3.14;
PI = 5;
A const declared as an array or object can be changed.
const ARRAY = [1,2,3,4,5];
ARRAY[2] = 9;
console.log(ARRAY);
If you do not want an object to be altered, use this code after your object is declared:
Object.freeze(OBJECTNAME);

ES6 Arrow Functions

Functions with only a return value can be simplified from this:
const myName = function() {
   const name = "John";
   return name;
}
to this:
const myName = () => {
   const name = "John";
   return name;
to this:
const myName = () => "John";
